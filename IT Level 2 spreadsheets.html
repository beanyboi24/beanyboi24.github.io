IT Level  2 spreadsheets

  

IT Level 2 spreadsheets was using a lot more formulas such as lookups, if statement, Macro buttons, lists.

IF statements and lookups allow the spreadsheet to automatically update when you choose an option.

Lookups allow you to make a table in order to A to Z otherwise it will not work properly.

IT Level 2 spreadsheets was a bit harder because some of the formulas was a bit harder to learn such as an if statment.
